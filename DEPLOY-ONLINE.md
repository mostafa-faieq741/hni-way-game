# HNI Way — Going Online (first deploy)

> دليل مبسّط لرفع اللعبة + الباك‑إند على النت مجاناً. مفيش حاجة فيها معقّدة — معظمها ضغطات.

## الفكرة في سطر
ثلاث حتت مجانية بتشتغل مع بعض:
- **GitHub** — يحفظ الكود.
- **Vercel** — يرفع اللعبة + الـ API على لينك عام (بيشغّل فولدر `api/` كوظائف تلقائياً).
- **Supabase** — قاعدة البيانات (المشاريع + تقدّم كل لاعب).

---

## الخطوات

### 1) الكود على GitHub
- اعمل repo جديد وارفع المشروع عليه.
- ده بيخلّي Vercel يسحب الكود منه ويرفع لوحده (وأي تعديل بعدين بيتنشر تلقائي).

### 2) قاعدة البيانات (Supabase)
1. <https://supabase.com> → **New project** (اختار region قريب من المستخدمين، وحط باسوورد للقاعدة).
2. **SQL Editor** → الصق محتوى `api/data/schema.sql` → **Run**. ده بيعمل جداول `projects` و`players` و`game_state`.
3. **Project Settings → API** → انسخ:
   - **Project URL** → هيتحط في `SUPABASE_URL`
   - **service_role** key → هيتحط في `SUPABASE_SERVICE_KEY` (سرّي — للسيرفر بس، مايتحطّش في المتصفّح)

### 3) الرفع على Vercel
1. <https://vercel.com> → **New Project** → اختار الـ repo بتاع GitHub.
2. Vercel بيتعرّف على Vite لوحده: Build = `npm run build`، Output = `dist`. فولدر `api/` بيتشغّل كوظائف.
3. في **Environment Variables** ضيف:
   - `SUPABASE_URL`
   - `SUPABASE_SERVICE_KEY`
   - (بعدين لو LMS) `TALENTLMS_DOMAIN` و`TALENTLMS_API_KEY`
4. **Deploy** → يطلعلك لينك عام (زي `hni-way.vercel.app`).

### 4) تأكيد إنه شغّال
- افتح اللينك → اللعبة تحمّل.
- افتح `اللينك/api/health` → المفروض يقول `{"ok":true,"storage":"supabase"}`.
- ادخل **Admin → Manage Projects** → ضيف بروجكت Published → اللاعبين يشوفوه فوراً.

## ملاحظات مهمة

- **خطوة كود صغيرة قبل الرفع:** الـ API دلوقتي مكتوب كسيرفر Express للتجربة المحلية (`api/server.js`). عشان يشتغل كوظائف على Vercel محتاج محوّل بسيط (ملفات وظائف تحت `api/`). **أنا بعملها** وقت ما نقرّر نرفع — مش خطوة عليك.
- **الـ proxy بتاع `/api`** اللي في `vite.config.js` للتجربة المحلية بس؛ على Vercel الـ API بيكون على نفس الدومين تلقائياً.
- **بدون LMS بيشتغل عادي:** اللاعب يدخل باسمه/إيميله وله هوية يحفظ ويرجّع بيها. الـ LMS اختياري ويضيف هوية تلقائية + تتبّع رسمي للكورس.

## بعد الرفع (الخطوة الجاية)
ربط **الحفظ/الاستكمال** لكل لاعب على Supabase عشان يرجع يكمّل من أي جهاز — يدعم الدخول البسيط + LMS مع بعض. نوصّلها لما تجهز.
